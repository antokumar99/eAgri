## ⚙️ Setup

```bash
git clone https://github.com/your-username/eagri.git
cd eagri
flutter pub get   # or npm install
flutter run
```

Create a `.env` file:
```env
WEATHER_API_KEY=your_key
SSLCOMMERZ_STORE_ID=your_store_id
FIREBASE_API_KEY=your_key
```
